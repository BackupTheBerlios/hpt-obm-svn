Charger des images

Pour insérer une image dans une page web, il faut d'abord la charger sur le serveur à distance.

Pour charger une image dans le dossier courant, il faut cliquer sur le bouton Charger Image à partir du panneau droit. Dans la fenêtre instantanée qui est affichée, il faut saisir le chemin local du fichier ou utiliser le bouton Parcourir pour naviguer parmi les dossiers et sélectionner le fichier.

Si l'on désire redimensionner l'image, il faut choisir une option parmi celles affichées dans la liste déroulante qui contient quelques dimensions prédéfinies. Pour maintenir la taille originelle de l'image, il faut sélectionner “Laisser Image”. On peut également saisir les dimensions qu'on désire en sélectionnant l'option “Personnalisé”. Le redimensionnement est fait en maintenant le taux, tel que l'image rentre dans un rectangle ayant les dimensions spécifiés. Cela veut dire que, parfois, une des dimensions sera recalculée pour s'accorder à la valeur de l'autre.

Finalement, cliquez sur le bouton Chargement pour confirmer votre sélection.
