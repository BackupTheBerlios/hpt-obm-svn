Nettoyer Code HTML

Lorsqu'on insère un ou plusieurs paragraphes qui sont déjà formatés par un autre éditeur de texte (par exemple Microsoft Word ou OpenOffice), la taille du code HTML augmentera considérablement. Ce phénomène est dû aux méthodes d'export utilisées par les deux éditeurs, qui insèrent beaucoup de balises inutiles dans le code HTML. Les utilisateurs de KTML3 ont la possibilité d'enlever ces balises et nettoyer ainsi le contenu inséré en utilisant l'option "Nettoyer Code HTML" et en appliquant ensuite les fonctions de traitement de KTML3. Le résultat est un code plus net et précis.
