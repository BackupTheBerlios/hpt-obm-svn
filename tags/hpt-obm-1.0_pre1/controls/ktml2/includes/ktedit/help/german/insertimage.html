Bild einfügen



Mit der Schaltfläche "Bild einfügen" können Sie Bilder in Ihre Seite einfügen. Nach dem Drücken der Schaltfläche wird von KTML ein Fenster mit einer Auswahl an Bildern anzeigt. Sie können auch weitere Bilder hochladen bzw. löschen. Beachten Sie beim Löschen der Bilder, dass diese bei keiner Internet-Seite mehr eingesetzt werden! Bilder können grundsätzlich in den von KTML editierbaren Bereich eingefügt werden. Löschen Sie ein Bild im editierbaren Bereich, wird dies nicht vom Server gelöscht, sondern nur von der Seite, die Sie gerade bearbeiten.
