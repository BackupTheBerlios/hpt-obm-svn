Wissel Edit Modus – Laat Code / Design zien

Omdat KTML3 een WYSIWYG tekst editor voor HTML documenten is, was het belangrijk om een functie te integreren die geavanceerde gebruikers in staat stelt het document direct in HTML formattering op te maken. Daarom heeft KTML3 editor twee opmaak modussen: een tekst editor wat het uiteindelijke resultaat laat zien, en een HTML editor. Om te schakelen/wisselen tussen deze twee modussen, klikt u op de "Laat Code/Design zien" knop in de KTML3 werkbalk.
