Configuration du Tableau

Après avoir cliqué sur le bouton Insérer Tableau de la barre d'outils, KTML3 ouvrira une fenêtre instantanée pour aider à la configuration du tableau:

  • Rangées – saisir le nombre de rangées; la valeur par défaut est 3;
  • Colonnes – saisir le nombre de colonnes; la valeur par d éfaut est 3;
  • Bordure – saisir l'épaisseur de la bordure du tableau; la valeur par défaut est 1.
