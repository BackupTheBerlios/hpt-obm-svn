Link einfügen



Sie können zu Bildern und zum Text Links hinzufügen. Links können zu externen und internen Seiten erstellt werden, weiters auch zu Dateien und zu Ankern auf der selben Seite. Um einen Link einzufügen, müssen Sie einen Text oder ein Bild markieren und dann auf die Schaltfläche "Link einfügen" klicken.

Wenn Links durch Kopieren&Einfügen (Cut&Paste) oder während des Schreibens automatisch eingefügt werden, können Sie im Eigenschafts-Inspektor die Links bei Bedarf korrigieren. Wenn Sie einen Link markieren, wird das Linkpanel im Eigenschafts-Inspektor automatisch angezeigt.

Links zu Ankern erstellen
Diese Art von Link referiert auf einen Text oder ein Bild innerhalb der selben Seite. Um einen solchen Link zu erstellen, müssen Sie zuerst einen Anker erstellen:

  • Text oder Bild markieren
  • die Schaltfläche "Link einfügen" klicken
  • einen Linknamen (Ankerpunkt) in den Eigenschafts-Inspektor schreiben

Als nächstens einen Links zum Anker erstellen:

  • zu verlinkenden Text oder Bild markieren
  • die Schaltfläche "Link einfügen" klicken
  • Schreiben Sie "#Anker" in das Linkfenster vom Eigenschafts-Inspektor, wobei Sie "Anker" durch den Linknamen (Ankerpunkt) ersetzen müssen.(Den zuvor eingegebenen). Beachten Sie, dass Sie keine Anführungszeichen im Linkfenster eingeben.

Benutzen Sie immer "#" wenn Sie einen Link zu einem Ankerpunkt setzen möchten.
