Text ein/ausrücken.



Text einrücken
Im KTML3 ist auch die Funktion des Text ein/ausrückens intergriert. Die Größe des Abstandes ist im Editor festgelegt und kann nicht geändert werden. Um Text einzurücken, brauchen Sie nur auf die Schaltfläche "Einrücken" drücken. Bei jedem Betätigen wird der Text um eine Stufe eingerückt.

Text ausrücken
Dieser Befehl bewirkt das Gegenteil, der Text wird bei jedem Betätigen der Schaltfläche ausgerückt.
