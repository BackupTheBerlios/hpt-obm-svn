Dateien hochladen

Um einen Dateidownloadlink zu platzieren, sollten Sie die Datei zuerst auf den Server hochladen.

Benutzen Sie das "Durchsuchen"-Fenster, um eine Datei auf den Server zu laden. Sie können folgende Dateiformate hochladen: bmp, gif, jpeg, jpg, png, pdf, doc, rtf, txt, mp3, mpg, mpeg, avi, wav, mov, qt, rm.

Alternativ können Sie auch direkt den Pfad von Ihrer Festplatte angeben, um eine Datei hochzuladen.

Abschließend drücken Sie die "Upload"-Schalftlfläche, um Ihre Auswahl zu bestätigen, und die Datei hochzuladen.
