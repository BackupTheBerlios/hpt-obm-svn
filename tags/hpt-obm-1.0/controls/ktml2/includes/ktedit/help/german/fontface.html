Schriftarten ändern



Der KTML Editor erlaubt es Ihnen, die Schriftarten aus dem Dropdown-Menü im Editor zu wechseln. Die Standart-Schriftart ist: Times New Roman, Times, serif.

Eine sehr nützlich Funktion von KTML ist auch die automatische Anzeige der Schriftart, um einen Text einer Schriftart zuzuordnen. Markieren Sie den Text, und die Schriftart wird automatisch im Schriftart-Dropdown angezeigt. Besteht ein Text aus mehreren Schriftarten, wird im Dropdown-Menü keine Schriftart angezeigt. Um den Text in eine einheitliche Schriftart darzustellen, markieren Sie den gesamten Text und wählen die Schriftart aus.

Sollten CSS-Stile im Dokument importiert werden, erscheinen diese "zusätzlichen" Schriftarten auch im Dropdown-Menü..
