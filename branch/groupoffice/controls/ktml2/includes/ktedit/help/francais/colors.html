Appliquer les couleurs

La palette de couleurs permet la sélection d'une couleur pour la police ou pour le surlignage du texte. Une palette de couleurs est une fenêtre qui affiche une large rangée de couleurs. On a aussi la possibilité d'introduire la valeur hexadécimale de la couleur. Cela est très utile pour les utilisateurs qui ont besoin d'une nuance précise.
