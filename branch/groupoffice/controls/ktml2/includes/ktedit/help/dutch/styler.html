CSS Stijlen toepassen



De “Cascading Style Sheets” (CSS) zijn een methode om stijl aan te brengen in HTML wat een grote stap voorwaards is ten opzichte van inline "stijl" declaraties, wat meer flexibiliteit geeft.

De stijlen van kunnen toegepast worden op elk type selectie (tekst, afbeelding, tabel). Indien u de CSS stijl voor een tekst wenst vast te zetten of veranderen, kunt u de tekst selecteren (of enkel een gedeelte van de tekst waar u de CSS voor in wenst te stellen) of u plaatst de cursor binnen de tekst en u kiest dan een CSS stijl van het CSS uitklap menu in de werkbalk. De stijl zal toegepast worden op het geselecteerde gedeelte van de tekst. Indien u enkel de cursor binnen de tekst heeft geplaats, en deze tekst heeft nog geen stijl definitie (de CSS staat nog op geen), zal de stijl ingesteld worden, inclusief voor alle tekst in de paragraaf welke nog geen stijl definitie heeft. Indien de tekst die in de paragraaf staat wel reeds een CSS definitie heeft, zal de nieuwe stijl toegepast worden, en ook op alle tekst in de paragraaf welke binnen de zelfde CSS definitie vallen (een SPAN tag). Dit is een zeer bruikbare optie, omdat de gebruiker geen tijd hoeft te besteden aan tekst selecties en aan het identificeren van stijlen.

Notitie: Om tot een goed resultaat te komen dient de gebruiker de CSS stijlen goed en consequent toe te passen op de tekst, in plaats van het continue aanpassen van en veranderen van CSS stijlen op verschillende stukken tekst. Dit omdat de HTML code te gecompliceerd wordt om de visuele restultaten goed naar voren te laten komen. Om de CSS stijlen goed te managen, kunnen gebruikers tevens de Tag Selectie gebruiken om snel en eenvoudig een aantal SPAN tags (corresponderend tot enkele reeds geplaatste stijlen) te verwijderen van de code.
