Tabellenoptionen

Wenn im KTML-Editor eine Spalte innerhalb der Tabelle ausgewählt ist, erscheint der Zellen-Inspektor (in der Optionenleiste kann auch <td> angeklickt werden). Neben Zellenoptionen kann dieser Inspektor auch eine Fülle von anderen Tabellenoptionen bearbeiten.

Einzelne Spalten vergrößern
Klicken Sie auf die Schaltfläche "Spaltenvergrößerung", um die Spalte zu vergrößern. Jeder Klick vergößert die Spaltenbreite.

Spalten einrücken
Der Zellenabstand wird bei jedem Klick um eine Zellenbreite eingerückt. Ist die Tabelle nur eine Spalte breit und Sie benützen diesen Befehl, erscheint folgende Fehlermeldung: “Zelle kann nicht verkleinert werden”.

Einzelne Reihen vergrößern
Klicken Sie auf die Schaltfläche "Reihenvergrößerung", um die Reihe zu vergrößern. Jeder Klick vergößert die Reihe.

Reihe einrücken
Der Zellenabstand wird bei jedem Klick um eine Zellenhöhe eingerückt. Ist die Tabelle nur eine Reihe hoch und Sie benützen diesen Befehl, erscheint folgende Fehlermeldung: “Zelle kann nicht verkleiner werden ”.

Spalte einfügen
Ein weiters Hightlight von KTML ist die Möglichkeit bei einer Tabelle, weitere Spalten einzufügen. Plazieren Sie die Textmarke in einer Zelle und drücken Sie diese Schaltfläche, um linksgerichtet eine Spalte einzufügen.

Spalte löschen
Sie können auch eine existierende Spalte in einer Tabelle löschen. Markieren Sie diese Spalte und drücken auf die Schaltfläche "Spalte löschen", um diesen Befehl auszuführen.

Reihe einfügen
Der Befehl ist sehr ähnlich mit dem Befehl "Spalte einfügen". Plazieren Sie die Textmarke in einer Zelle und drücken Sie die Schaltfläche, um unterhalb der Auswahl eine Reihe einzufügen.

Reihe löschen
Sie können auch eine existierende Reihe in einer Tabelle löschen. Markieren Sie diese Reihe und drücken auf die Schaltfläche "Reihe löschen", um diesen Befehl auszuführen.
