Link Inspektor

Sie müssen zuerst einen Text markieren, um einen Link mit der Linktaste einfügen zu können. Wenn Sie den Link mit der Maus markieren, werden im Link Inspektor die Eigenschaften angezeigt.


  • Ziel – in dem Dropdown-Menü wird das Linkziel angegeben. Wählbare Werte sind: normal, neues Fenster, dieses Fenster, dieser Frame, übergeordneter Frame:
    • normal : Standarteinstellung
    • neues Fenster : es wird ein neues Browser-Fenster geöffnet
    • dieses Fenster : der Inhalt wird im selben Fenster angezeigt (jedoch nicht im Frame!)
    • dieser Frame : der Inhalt wird im selben Rahmen (Frame) angezeigt.
    • übergeordneter Frame : der Inhalt wird im übergeordneten Rahmen (Frame) angezeigt.
  • Name – gibt den Linknamen an;
  • Href– das Textfeld, indem Sie die den richtigen Link (URL) angeben.
  • Durchsuchen – diese Taste öffnet den KTML Datei Navigator, er erlaubt Ihnen auf Dateien zu verlinken
  • Link löschen – diese Taste löscht den markierten Link
