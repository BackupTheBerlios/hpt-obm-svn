HTML bereinigen



Wenn Sie Zeilen oder Text einfügen, die bereits durch ein anderes Programm formatiert wurden (zB Microsoft Wort, OpenOffice, usw.), kann sich die Größe des HTML-Codes beträchtlich ändern. Dieses Phänomen liegt an den Exportmethoden der verschiedenen Anwendungen, die häufig unbrauchbaren oder zusätzlichen HTML-Code verwenden. Sie als Benutzer von KTML3 haben die Möglichkeit, den HTML-Code von diesen zusätzlichen Einträgen zu bereinigen. Drücken Sie dazu die Schaltfläche mit "HTML bereinigen". Das Resultat ist ein bereinigter und ein genauerer Code.
