Appliquer des feuilles de styles CSS



Les feuilles de style CSS ( “Cascading Style Sheets”) sont une méthode d'implémenter les styles dans HTML qui apporte une grande flexibilité.

Les feuilles de style CSS peuvent être appliqués à toute sorte de sélection (texte, image, tableau). Lorsqu'on désire appliquer une feuille de style à un texte qui n'a aucun style, le texte doit être d'abord sélectionné. De l'autre côté, si on a appliqué une feuille de style à un texte et on veut la changer, il est suffisant de positionner le curseur à l'intérieur du texte et de sélectionner la nouvelle feuille. La feuille de style sera appliquée au texte entier. Cela est très utile puisqu'on ne perd plus du temps en sélectionnant le texte et en identifiant les styles.
