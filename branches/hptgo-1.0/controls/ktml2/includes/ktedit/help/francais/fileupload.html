Charger des fichiers

Pour créer un lien vers un fichier, on doit d'abord charger le fichier désiré sur le serveur à distance.

Il faut saisir le chemin local du fichier ou utiliser le bouton Parcourir pour naviguer parmi les dossiers locaux et pour sélectionner le fichier. On peut charger seulement les types de fichiers supportés: bmp, gif, jpeg, jpg, png, pdf, doc, rtf, txt, mp3, mpg, mpeg, avi, wav, mov, qt, rm.

Finalement, il faut cliquer sur le bouton Chargement pour confirmer la sélection.
