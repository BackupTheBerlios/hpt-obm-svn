Traitement texte

Couper
Cette commande actionne sur le texte/image sélectionné. Après l'avoir exécutée, le texte/image sélectionné est supprimé du document et placé dans le Presse Papier d'où il peut être collé soit dans la région éditable de KTML3, soit dans un autre éditeur. Cette commande est accessible à travers le bouton Couper ou en utilisant les touches de raccourcis “Ctrl+X”.
Note: Lorsqu'on utilise le navigateur Mozilla, la commande Couper n'est pas accessible à partir de la barre d'outils, mais seulement en utilisant les touches de raccourcis.

Copier
Cette commande actionne sur le texte/image sélectionné. Après l'avoir exécutée, le texte/image sélectionné est copié dans le Presse Papier d'où il peut être collé soit dans la région éditable de KTML3, soit dans un autre éditeur. Cette commande est accessible à travers le bouton Copier ou en utilisant les touches de raccourcis “Ctrl+C”.
Note: Lorsqu'on utilise le navigateur Mozilla, la commande Copier n'est pas accessible à partir de la barre d'outils, mais seulement en utilisant les touches de raccourcis.

Coller
Cette commande actionne sur le texte/image sélectionné ou sur la position courante du curseur. Après l'avoir exécutée, le texte/image du Presse Papier est collé dans la région éditable de KTML3.Si un autre texte/image est déjà sélectionné en cliquant sur le bouton Coller, le texte/image respectif sera remplacé par celui du Presse Papier. Cette commande est accessible à travers le bouton Coller ou en utilisant les touches de raccourci “Ctrl+V”.
Note: Lorsqu'on utilise le navigateur Mozilla, la commande Coller n'est pas accessible à partir de la barre d'outils, mais seulement en utilisant les touches de raccourcis.
