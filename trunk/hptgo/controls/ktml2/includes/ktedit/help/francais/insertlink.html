Insérer Lien



Les liens peuvent être appliqués soit aux sélections de texte, soit aux images. On peut lier la page courante aux autres pages du site, aux sites externes, aux fichiers téléchargeables ou aux ancres dans la même page. Pour insérer un lien, il faut sélectionner le texte/image désiré et cliquer sur le bouton Insert Link de la barre d'outils.

Il faut d'abord saisir un texte dans la région éditable (même si cela représente une adresse URL valide), cliquer ensuite sur le bouton Insérer Lien et définir l'adresse correcte dans le panneau Configuration Lien. Ce panneau sera affiché automatiquement en dessous de la région éditable lorsqu'on clique sur le bouton "Insérer Lien". Le panneau Configuration Lien et affiché également lorsqu'on clique à l'intérieur d'un texte qui représente un lien.

Créer Ancre
On peut également utiliser la fonctionnalité "Insérer Lien" pour créer une ancre. Ce type de lien rédirectionne le visiteur à une autre section de la même page web. Il faut d'abord créer l'ancre vers laquelle le lien pointera:

  • sélectionner le texte;
  • cliquer sur le bouton Insérer Lien;
  • saisir le nom de l'ancre dans le champ Nom du panneau Configuration Lien.

Ensuite, il faut créer le lien proprement-dit:

  • sélectionner le texte qui sera utilisé pour le lien;
  • cliquer sur le bouton Insérer Lien;
  • saisir "#ancre" dans le champ Lien du panneau Configuration Lien, où “ancre” doit être remplacé par le nom donné à l'ancre.

Il faut toujours se rappeler d'utiliser le symbole #, suivi par le nom de l'ancre lorsqu'on crée une ancre.
